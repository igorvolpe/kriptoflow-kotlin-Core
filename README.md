# 📊 KriptoFlow - Market Monitor

Projeto desenvolvido para consolidar fundamentos avançados de **Kotlin**, com foco em arquitetura limpa e alta performance.

## 🛠 Tecnologias e Conceitos Aplicados

- **Kotlin Core:** Uso de `Data Classes`, `Imutabilidade (val)` e `Scope Functions`.
- **Null Safety:** Implementação rigorosa para evitar NullPointerExceptions usando `Safe Calls` e `Elvis Operator`.
- **Concorrência:** Gerenciamento de threads com **Kotlin Coroutines**.
- **Functional Programming:** Manipulação de coleções com filtros e mapeamentos de dados.

## 🏗 Estrutura do Projeto
O projeto simula um ambiente de P&D onde a precisão dos dados e a segurança de tipos são prioridades.
